# 西安交通大学港澳台合办活动电子问卷

这是一个单文件静态网页问卷，用于统计西安交通大学与港澳台学校合办活动。

## 使用方式

发布到 GitHub Pages 后，访问：

https://Kryr1ch.github.io/xjtu-hmt-questionnaire/

填写者可以录入一个活动下的多个参与学校联系人，并在页面内导出 CSV。

## 字段

- 活动名称：必填
- 活动链接：选填
- 参与学校：必填
- 学校联系人：必填
- 联系人身份：学生 / 老师，必填
- 邮箱 / 电话：二选一必填

## 注意

当前版本为本地浏览器保存和 CSV 导出，不会自动上传填写结果到服务器。
